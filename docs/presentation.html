Vertex Shader — GPU Skinning

Each vertex blends up to 4 bone transforms by their skin weights:

mat4 skin = weights.x * bones[boneIDs.x]
           + weights.y * bones[boneIDs.y]
           + weights.z * bones[boneIDs.z]
           + weights.w * bones[boneIDs.w];
gl_Position = Proj * View * Model * skin * vec4(aPos, 1.0);

Fragment Shader — Blinn-Phong Shading

  • Ambient 0.15 — prevents pure black shadows
  • Diffuse 0.8 — Lambert: max(N·L, 0)
  • Specular 0.2 — half-vector: (N·H)³²
  • Albedo from glTF texture (stb_image)
  • Normals via inverse-transpose of skin matrix

Geometric Objects & Transformations

Bone Hierarchy = Scene Graph

Each bone stores a parent index and a local TRS (Translation · Rotation · Scale) matrix. A single forward pass builds global world transforms:

global[i] = global[parent[i]] × local[i]

Skinning Matrix

Brings a vertex from bind-pose into animated world space:

M_skin[i] = global[i] × invBind[i]

invBind moves vertices back to bone-local space before animating. At t=0 the result must be identity — our validation check.

Animation Sampling & Blending

Keyframe Interpolation

  • Translation / Scale — linear LERP between keyframes
  • Rotation — SLERP (spherical linear) on unit quaternions — shortest arc, no gimbal lock
  • Binary search finds surrounding keyframes each frame

Animation Blending (N key)

  • Sample both clips at TRS level simultaneously
  • Blend α goes 0 → 1 over 0.4 s
  • Source clip time frozen; target starts from 0
  • SLERP rotations — no matrix lerp (causes shearing)

Why Not Matrix LERP?

Linearly interpolating two rotation matrices breaks orthogonality — the result shears and deforms the mesh. Blending must happen at TRS level before building the mat4.

Bind Pose Validation

At t = 0, M_skin[i] must equal identity for every bone (global = bind, so global × invBind = I). Press 0 to run the check and print the max error.

Buffers & Texture Mapping · Building 3D Models

VAO / VBO

Single interleaved buffer per mesh:

0 pos vec3
1 normal vec3
2 uv vec2
3 boneIDs ivec4
4 weights vec4

UBO — Bone Matrices

128 skinning mat4s in a std140 Uniform Buffer Object. A single glBufferSubData call uploads all bones per frame.

128 × 64 B = 8 KB per frame

SSBO (GL 4.3+) would allow unlimited bones — macOS caps at 4.1.

Albedo Texture

Loaded via stb_image, uploaded with mipmaps. Sampled in the fragment shader using the mesh's UV coordinates.

glTF 2.0 Format

Parsed with tinygltf. One file contains geometry, skeleton, inverse bind matrices, animation channels, and textures.

Viewing & Projection · Input & Interaction

Orbit Camera

Parameterised by yaw, pitch, radius, target. View matrix rebuilt from spherical coords each frame. 45° FOV perspective projection, aspect updated on resize.

gl_Position = Proj × View × rootTransform × skinnedPos

rootTransform handles glTF axis conventions (Z_UP nodes above the skeleton).

GLFW Input

  • LMB drag → orbit (yaw / pitch)
  • RMB drag → pan (translate target)
  • Scroll → zoom (radius)
  • Space → play / pause
  • N → smooth blend to next clip
  • J / L → frame-step scrub ±1/30 s

Dear ImGui Debug Panel

  • Play/pause · Reset · time scrub slider
  • Speed slider (0.05× – 8×)
  • Clip selector — click to crossfade
  • Blend progress bar
  • Bone overlay toggle
  • FPS + triangle counter

Keyboard Layout Fix

Used glfwSetCharCallback (Unicode codepoints) for +/− speed keys instead of key codes — works correctly on any keyboard layout.

Key Challenges & Solutions

🌀
Model tiny & upside-down
Non-joint ancestor nodes (Z_UP, Armature) carry orientation. Fixed: DFS walk accumulates their transforms into a rootTransform applied as uModel.
📦
layout(binding=0) compile error
OpenGL 4.1 doesn't support explicit UBO binding qualifiers in GLSL. Removed from shader; binding set via glUniformBlockBinding() at load time.
🦴
Skeleton overlay invisible
Depth test culled lines sitting inside the mesh. Fixed: glDisable(GL_DEPTH_TEST) for the bone draw pass, restored after.
🔀
Animation blending caused mesh shearing
Matrix-level lerp breaks rotation orthogonality. Solution: blend at TRS level — SLERP quaternion rotations, LERP translation and scale before building mat4.
⌨️
Speed keys broken on macOS keyboard layout
GLFW key codes differ by layout. Used glfwSetCharCallback which fires actual Unicode codepoints — layout-independent.
